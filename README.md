### 3. Contact Form
The contact form is a static HTML frontend. To make it functional, we recommend using a free service like **Formspree** or **EmailJS**.
- Go to [Formspree.io](https://formspree.io) and create a free form.
- Copy your unique endpoint URL.
- In `index.html`, find the `<form>` tag and paste it:
  `<form action="YOUR_FORMSPREE_URL" method="POST">`
